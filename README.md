# 棒読みちゃん 動画編集拡張プラグイン

棒読みちゃんに動画編集 (音声素材の作成など)に有用な拡張機能を追加するプラグインです。

## 動作確認環境
- 棒読みちゃん Ver0.1.11.0 β21
- .NET Framework 3.5

---

## 導入方法

1. GitHubの「Releases」から最新の `Plugin_VideoExtension.dll` をダウンロードします。
2. `BouyomiChan.exe` が存在するフォルダに、ダウンロードしたDLLファイルを配置します。
3. 棒読みちゃんを起動し、プラグインを有効化します。

---

## 機能

現在、以下の3つの機能を提供しています。

### 1. プリセット機能
よく使う声の設定を保存・管理できます。
- **追加**: パラメーター (声質・音量・速度・音程)を変更し、「＋ 追加」からプリセット名を入力して保存します。
- **適用**: プルダウンからプリセットを選択すると、設定が即座に反映されます。
- **更新**: 「↻ 更新」を押すと、選択中のプリセットに現在のパラメーターを上書き保存します。
- **削除**: 「－ 削除」を押すと、選択中のプリセットを削除します。

### 2. パラメーター数値入力
「✏️ 数値入力」を押すと、スライダーでは微調整しにくいパラメーターを直接数値で入力するためのウィンドウが表示されます。

### 3. 音声・テキスト同時書き出し
「💾 書出」を押すと、現在入力されているテキストの音声ファイル (.wav)と、テキストファイル (.txt)が同時に出力されます。
※初回利用時、または出力先を変更したい場合は、プラグインの設定画面から「ファイル保存先」を指定してください (存在しないフォルダが指定された場合には自動生成されます)。
